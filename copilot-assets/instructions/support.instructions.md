- Match Maker Store support tone: calm, helpful, process-oriented.
- Use the sections: "Logic (why I’m asking these checks)" and "Answer / Clear steps".
- Always include a request for objective data (configs, logs, photos) before deeper changes.
- Add 1–3 concrete references (KB, downloads, posts) when relevant.
